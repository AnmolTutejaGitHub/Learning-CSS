How can you use CSS to make a web page more accessible to users with disabilities?

Improving web accessibility for users with disabilities is essential to provide an inclusive user experience. CSS can contribute to accessibility in various ways:

1. Use Readable Fonts:

<style> body { font-family: Arial, sans-serif; font-size: 16px; } </style>

2. Provide Sufficient Line Height:

<style> body { line-height: 1.6; } </style>

These CSS styles contribute to a more accessible and readable webpage for all users, including those with disabilities.
