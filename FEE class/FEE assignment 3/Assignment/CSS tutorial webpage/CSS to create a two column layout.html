How can you use CSS to create a two-column layout?

To create a two-column layout using CSS, you can use the CSS Flexbox layout. Flexbox allows you to easily create a flexible layout with two columns. Here's a step-by-step example:

Create a CSS Rule for the Container:

<style> .container { display: flex; } </style>

Create CSS Rules for the Columns:

<style> .column { flex: 1; padding: 20px; border: 1px solid #ccc; margin: 10px; } </style>

Define HTML Structure for Two Columns:

<div class="container"> <div class="column"> Content for the first column. </div> <div class="column"> Content for the second column. </div> </div>

The CSS rules for the container and columns use Flexbox to create a two-column layout. Adjust padding, border, and margin as needed for your design.
