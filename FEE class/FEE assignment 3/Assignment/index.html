Types of CSS

CSS (Cascading Style Sheets) comes in three types:

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS : Inline CSS involves directly applying styles to individual HTML elements using the style attribute within the HTML tag. This style is specified directly within the HTML element and overrides any external or internal styles affecting the same element.


Internal CSS: Internal CSS involves placing CSS rules within the <style> element within the <head> section of an HTML document. The CSS rules defined in this section apply to the HTML elements within the same document.

External CSS : External CSS involves creating a separate CSS file with .css extension and linking it to the HTML document using the <link> element within the <head> section. The CSS rules defined in this external file apply to the HTML elements throughout the entire website or the specified pages.
